# AI Fire Guardian

AI Fire Guardian ek real-time fire detection system hai jo YOLOv8, Python aur OpenCV use karta hai.

## Features
- Real-time fire detection
- Webcam monitoring
- Fire detect hone par alarm
- Screenshot capture
- Telegram alert system

## Technologies
Python  
YOLOv8  
OpenCV  
Telegram Bot API  

## Run the project

Install dependencies:

pip install ultralytics opencv-python requests

Run command:

python fire_alarm.py
